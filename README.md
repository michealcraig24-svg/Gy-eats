# GY Eats - Find Cheap Food in Guyana

A modern web application helping Guyanese and visitors find affordable, delicious food across Georgetown, Linden, and New Amsterdam.

## 🍔 Features

- **50+ Restaurants** - Curated list of affordable food spots
- **Real Prices** - Actual GYD pricing from local vendors
- **Search & Filter** - Find by cuisine type, location, or price range
- **Interactive Menus** - View detailed menu items and prices
- **Hot Deals** - Daily specials and family meal deals
- **Price Guide** - Understanding local food costs
- **Responsive Design** - Works on desktop and mobile devices
- **Beautiful UI** - Modern dark theme with smooth animations

## 🚀 Quick Start

1. Clone the repository:
```bash
git clone https://github.com/michealcraig24-svg/Gy-eats.git
cd Gy-eats
```

2. Open `index.html` in your browser - no installation needed!

3. Or enable GitHub Pages:
   - Go to Settings → Pages
   - Select "main" branch as source
   - Visit: `https://michealcraig24-svg.github.io/Gy-eats/`

## 📱 Features Overview

### Search & Filter
- Search by restaurant name, cuisine, or location
- Filter by food type: Fast Food, Local Food, Street Food, Chinese
- Price range filters (Under $500)

### Restaurant Cards
- Rating and reviews count
- Price range indicator
- Operating hours
- Contact information
- Tags (budget, local-favorite, etc.)
- View menu and call buttons

### Hot Deals Section
- Featured daily specials
- Family meal packages
- Savings display

### Price Guide
- Street Food & Snacks: $200-$400
- Fast Food Meals: $500-$1,200
- Local Cook-up & Curry: $800-$1,500
- Seafood Specials: $1,200-$2,500

## 🔄 How It Works

1. **Load Restaurants** - All 12 restaurants load on page initialization
2. **Search** - Type to filter restaurants in real-time
3. **Filter** - Click filter chips to narrow by type or price
4. **View Menu** - Click "View Menu" button to see detailed menu items
5. **Contact** - Click "Call" button to phone the restaurant
